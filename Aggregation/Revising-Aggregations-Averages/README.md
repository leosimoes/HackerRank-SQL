# HackerRank - SQL - Aggregation - Revising Aggregations - Averages

Query the average population of all cities in CITY where District is California.


## Input Format
The CITY table is described as follows:

| Field       | Type        |
|-------------|-------------|
| id          | number      |
| name        | varchar(17) |
| countrycode | varchar(3)  |
| district    | varchar(20) |
| population  | number      |
