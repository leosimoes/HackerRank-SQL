# HackerRank - SQL - Aggregation - Japan Population

Query the sum of the populations for all Japanese cities in CITY. The COUNTRYCODE for Japan is JPN.


## Input Format
The CITY table is described as follows:

| Field       | Type        |
|-------------|-------------|
| id          | number      |
| name        | varchar(17) |
| countrycode | varchar(3)  |
| district    | varchar(20) |
| population  | number      |
