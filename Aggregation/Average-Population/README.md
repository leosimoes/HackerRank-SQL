# HackerRank - SQL - Aggregation - Average Population

Query the average population for all cities in CITY, rounded down to the nearest integer.


## Input Format
The CITY table is described as follows:

| Field       | Type        |
|-------------|-------------|
| id          | number      |
| name        | varchar(17) |
| countrycode | varchar(3)  |
| district    | varchar(20) |
| population  | number      |
