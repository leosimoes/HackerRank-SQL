# HackerRank - SQL - Aggregation - Population Density Difference

Query the difference between the maximum and minimum populations in CITY.


## Input Format
The CITY table is described as follows:

| Field       | Type        |
|-------------|-------------|
| id          | number      |
| name        | varchar(17) |
| countrycode | varchar(3)  |
| district    | varchar(20) |
| population  | number      |
